# Personagens - Imagens

Coloque as imagens dos personagens nesta pasta para que elas apareçam no site.
Os arquivos devem ter os seguintes nomes (conforme definido no código):

- Crono: `crono.jpg`
- Marle: `marle.jpg`
- Lucca: `lucca.jpg`
- Frog: `frog.jpg`
- Robo: `robo.jpg`
- Ayla: `ayla.jpg`
- Magus: `magus.jpg`

Você pode usar outros formatos (png, webp), mas precisará alterar o código em `pages/Characters.tsx` se fizer isso.
